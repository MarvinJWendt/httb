# httb

> HTTP bin

## Configuration

### Environment Variables

| Name         | Description                                                | Default  |
|--------------|------------------------------------------------------------|----------|
| `LISTEN`     | Address to listen to                                       | `:8080`  |
| `LOG_LEVEL`  | Log level (can be one of `debug`, `info`, `warn`, `error`) | `info`   |
| `LOG_FORMAT` | Log format (can be one of `logfmt`, `json`)                | `logfmt` |
